# 🎬 PlayTube Backend (Node.js + MongoDB)

A scalable backend for **PlayTube** — a YouTube-like video platform with built-in **tweet-style microblogging**, built using **Node.js, Express, and MongoDB**.

---

## 🚀 Overview

PlayTube backend powers a modern content platform where users can:

* Upload & stream videos
* Post tweets (short content updates)
* Interact via likes, comments, subscriptions
* Discover trending videos & posts

---

## 🧱 Tech Stack

* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB (Mongoose ODM)
* **Authentication:** JWT + bcrypt
* **File Storage:** Cloudinary
